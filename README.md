htpasswd-java-webapp
====================

## Apache htpasswd generator webapp

Übersimple htpasswd Apache/htpasswd generator webapp.

Can generate passwords in the following rutines:
* Base64+SHA-1
* MD-5
* Crypt(3)

The "default operation" will be Crypt(3)
